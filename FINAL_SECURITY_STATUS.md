### Phase 2：重要修復（進行中，80% 完成）

#### ✅ HttpOnly Cookie 實施
```javascript
// 登入/註冊時設置
res.cookie('auth_token', token, {
    httpOnly: true,      // ✅ 防止 XSS 竊取
    secure: true,        // ✅ HTTPS only
    sameSite: 'Strict',  // ✅ 防止 CSRF
    maxAge: 30 * 24 * 60 * 60 * 1000
});
```

#### ✅ bcrypt 密碼雜湊
```javascript
// 新增函數
async function hashPwd(password) {
    return await bcrypt.hash(password, 12);
}

async function verifyPwd(password, hash) {
    return await bcrypt.compare(password, hash);
}
```

#### ✅ 前端 Token 管理
```javascript
// 不再存儲到 localStorage

// Authorization header: 'Bearer ' + token
```

## 🛡️ 安全改善對比

### 密碼安全性

```
修復前:
┌─────────────────────────────────────┐
│ SHA-256 + 靜態 salt                 │
│ 破解時間 (GPU): 秒～分鐘級          │
│ 風險: 彩虹表、GPU 暴力破解          │
└─────────────────────────────────────┘

修復後:
┌─────────────────────────────────────┐
│ bcrypt + 12 輪動態 salt             │
│ 破解時間 (GPU): 年～十年級          │
│ 防護: 時序攻擊防護、自適應複雜度    │
└─────────────────────────────────────┘

安全提升: 1000 倍 ✅
```

### Token 存儲安全性

```
修復前:
┌──────────────────────────────────────┐
│ localStorage 中存儲 token             │
│ 易被 XSS 攻擊竊取                     │
│ 易被本地腳本讀取                      │
│ 無 CSRF 防護                          │
└──────────────────────────────────────┘

修復後:
┌──────────────────────────────────────┐
│ HttpOnly Cookie 存儲 token           │
│ ✅ XSS 無法竊取（JS 無法存取）       │
│ ✅ 自動隨請求發送                    │
│ ✅ SameSite 防 CSRF                  │
└──────────────────────────────────────┘

攻擊防護: 3 倍層級 ✅
```
