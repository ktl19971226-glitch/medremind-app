# 審查備註

藥護家是用藥提醒與家人照護工具，不提供醫療診斷、治療或處方建議。AI 藥袋辨識僅用於輔助輸入文字，使用者仍需自行確認藥袋、處方、醫師或藥師資訊。

測試方式：
1. 開啟 App 後建立或使用裝置帳號。
2. 新增一筆用藥提醒。
3. 測試服藥打卡與通知中心。
4. 可使用家人邀請碼流程測試家人照護。
5. 可使用相機或相簿測試藥袋辨識入口。
6. 可到設定頁測試問題回報、資料匯出與帳號刪除。
